## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn/ui + Radix
- **AI Integration**: Gemini (via Google Generative AI)
- **Deployment**: Vercel-ready
- **Export**: Pure HTML + Tailwind (no vendor lock-in)

## 📦 Installation & Local Development

```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/squpage.git
cd squpage

# 2. Install dependencies
npm install

# 3. Create .env.local and add your keys
cp .env.example .env.local

# Add these:
# GEMINI_API_KEY=your_gemini_key_here
# (Get free key from https://aistudio.google.com)

# 4. Run development server
npm run dev
